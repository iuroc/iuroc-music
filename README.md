# vanjs-start

> 快速创建 Van.js 项目开发环境

## 使用方式

```bash
git clone https://github.com/iuroc/vanjs-start.git
npm install
```

- 启动开发环境（每条命令各创建一个终端）
    ```bash
    npm run dev:frontend
    npm run dev:backend:tsc
    npm run dev:backend:node
    ```
- 启动生产环境
    ```bash
    npm run build
    npm run start
    ```
